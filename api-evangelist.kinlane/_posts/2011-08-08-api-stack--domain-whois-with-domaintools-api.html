--- layout: post title: 'API Stack - Domain Whois with DomainTools API' source: http://apievangelist.com/2011/08/08/api-stack--domain-whois-with-domaintools-api/ domain: apievangelist.com image: http://kinlane-productions.s3.amazonaws.com/api-evangelist-site/blog/domain-tools-logo.png --- The DomainTools API provides research and monitoring web services for domains.

Using DomainTools you can perform whois lookup, history, reverse IP, name server reports, domain suggestion and search.

The DomainTools API is RESTful, using HMAC authentication and returns XML, JSON and HTML responses.

DomainTools charges a monthly fee, and per call charge for each of the domain services.

DomainTools API for domain related services is a perfect tool for the API stack
