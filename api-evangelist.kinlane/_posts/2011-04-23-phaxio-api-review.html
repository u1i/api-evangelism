--- layout: post title: "Phaxio API Review" url: 'http://apievangelist.com/2011/04/23/phaxio-api-review/' image: '' --- Phaxio lets developers build faxing capabilities into systems with a simple API.

The Phaxio API provides a RESTful API, using production and test application keys for authentication, with all API responses in JSON.

The entire application is centered around the API, and has no language kits or samples code currently.
