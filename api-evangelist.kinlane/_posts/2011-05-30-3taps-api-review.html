--- layout: post title: '3Taps API Review' source: http://apievangelist.com/2011/05/30/3taps-api-review/ domain: apievangelist.com image: --- 3taps is a data platform that aggregates searches and posting to multiple data platforms into single interface.

3taps API provides a RESTful interface with app and keys for authentication, provided by Mashery. The platform provides language libraries in the following formats:JavaScript, Java, PHP, Perl, Objective C, Python, Ruby.
