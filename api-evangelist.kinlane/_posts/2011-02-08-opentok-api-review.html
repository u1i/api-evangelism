--- layout: post title: 'OpenTok API Review' source: http://apievangelist.com/2011/02/08/opentok-api-review/ domain: apievangelist.com image: --- OpenTok is a free set of APIs from TokBox that enables websites to integrate live, group video communication into their web sites and applications.

The OpenTok provides a RESTFul API using an application key and user token authentication.

Server-side libraries available in Java, PHP, Python, and Ruby.
