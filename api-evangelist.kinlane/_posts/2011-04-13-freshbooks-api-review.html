--- layout: post title: 'Freshbooks API Review' source: http://apievangelist.com/2011/04/13/freshbooks-api-review/ domain: apievangelist.com image: --- Freshbooks provides time tracking, expense and invoicing Software as a Service (SaaS).

The Freshbooks API is a RESTful API, using OAuth for authentication. The API provides sixteen endpoints, providing the same functionality through the API, as whats available through the web application.

The Freshbook API community provides Ruby, Python, PHP, CakePHP, Perl, JavaScript, ActionScript, Java, and .NET sample code libraries.
