--- layout: post title: "APIs Being Used for Public Health" url: 'http://apievangelist.com/2011/04/28/apis-being-used-for-public-health/' image: '' --- Foursquare just highlighted an innovative use of their API, that alerts people to public health issues.

DontEat.at is a service which merges city health code records with the Foursquare API to warn people via text message, when they check into a restaurant that has recorded health code violations.

An excellent example of the power of APIs for mashing up data and providing real-time value that can even be used for the public good.
