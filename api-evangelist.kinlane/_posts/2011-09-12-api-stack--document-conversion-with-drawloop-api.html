DrawLoop provides a SOAP and REST API for conversion of documents in various formats, using a User ID and Password combination for authentication.

Using the DrawLoop LOOP API, developers can convert documents in the following formats: Microsoft Word, Microsoft PowerPoint, Microsoft Excel, Adobe PDF, XML, Microsoft Visio, JPEG, PNG, BMP, URLs.Â
