--- layout: post title: 'Flattr - API Launch' source: http://apievangelist.com/2011/01/28/flattr--api-launch/ domain: apievangelist.com image: --- Flattr the social micro-payment site, made their API public today.

The API comes in two flavors.
  • Javascript API
  • REST API
Flattr provides application registration and authentication with oAuth.

They also have a Flattr developers forum to support the community.
