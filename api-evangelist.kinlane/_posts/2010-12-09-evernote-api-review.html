--- layout: post title: 'Evernote API Review' source: http://apievangelist.com/2010/12/09/evernote-api-review/ domain: apievangelist.com image: --- Evernote is a text, voice and image note-taking software. A software as a service , Evernote provides web-based, desktop and mobile clients for accessing the service.

The Evernote API is delivered using its own proprietary protocol called Evernote Data Access and Management (EDAM) using OAuth for authentication for web applications. The web service provides a Mac, Windows, and mobile specific tool-kits to integrate with the platform.
