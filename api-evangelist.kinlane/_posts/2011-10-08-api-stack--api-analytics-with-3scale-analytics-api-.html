3scale provides scalable API Infrastructure as a Service to control, manage and monetize the distribution of data, content or services for devices, mobile and web applications.

The 3Scale Analytics API lets you pull out usage statistics for your API, either for the whole service or for each application, allowing you to integrate stats information into your own internal dashboards or provide enhanced dashboards for your API users.

The 3Scale Analytics API provides a RESTful API, using a key for authentication, and responses in an XML format.

3Scale provides a freemium account with 50K calls per day, and other packages for higher volume calls.

The 3Scale Analytics API for API analytics is a perfect tool for the API Stack.
