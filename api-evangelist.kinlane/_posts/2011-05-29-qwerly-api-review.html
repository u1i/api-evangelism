--- layout: post title: "Qwerly API Review" url: 'http://apievangelist.com/2011/05/29/qwerly-api-review/' image: '' --- Qwerly provides a single data API for accessing social network profiles across multiple networks including Twitter, Facebook, and Twitter.

Qwerly only offers a RESTful API, using key and app authentication provided by Mashery. Qwerly provides Node.js, Kynetx, Zend PHP, Python langugage libraries.
